# raspi-camera-s3 [![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat)](LICENSE)
Take photos and upload those to AWS S3.

![image](./images/20160926070548.jpg)

```
$ AWS_ACCESS_KEY_ID="" \
  AWS_SECRET_ACCESS_KEY="" \
  AWS_DEFAULT_REGION="" \
  AWS_S3_BUCKET="" \
  ./raspi-camera-s3
```
